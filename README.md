# 🤖 Rule-Based Chatbot using Python

## 📌 Project Description
This is a simple rule-based chatbot developed using Python.  
It interacts with users based on predefined rules and responses.  
It can also show current date and time.

This project is created as part of the **CodeAlpha Python Programming Internship**.

---

## 🚀 Features
- Greeting responses (Hi, Hello, etc.)
- Shows current Date
- Shows current Time
- Predefined chatbot replies
- Exit option using "bye"

---

## 🛠️ Technologies Used
- Python 3
- Random Module
- Datetime Module
